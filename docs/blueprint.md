# **App Name**: EduTrack360

## Core Features:

- Landing Page: Landing page with brief information about Edutrack360 and login options.
- Login Page: User login page with role selection (admin, staff, student), and an email/password form that uses the ID to locate email
- Payment Section: Payment section: Integrate Flutterwave mobile money for Zambia (Airtel, MTN).
- Course Display and Linking: Course list display showing the name and year of the course, filtering courses and linking them to programmes.
- Academic Calendar: Academic calendar displays, download functionality, event display, holiday API integration.
- Profile Management: User profile updates, allowing picture upload and number edits.
- Notification System: Notification summary, showing the total of unread notification, as a tool, highlighting relevant user actions

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and trustworthy feel, evoking stability and intelligence which reflects the educational nature of the platform.
- Background color: Light blue-gray (#E8EAF6), offering a clean, neutral backdrop that reduces eye strain and enhances content visibility.
- Accent color: Vibrant orange (#FF9800) to highlight calls to action, important notifications, and interactive elements, ensuring key features are easily noticeable.
- Headline font: 'Space Grotesk' sans-serif, providing a modern, tech-forward feel for headings.
- Body font: 'Inter' sans-serif for body text to ensure readability and a clean user experience.
- Use modern and simple icons for navigation and features to enhance usability.
- Clean and structured layout with a focus on intuitive navigation.
